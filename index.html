PHP-Framework-Benchmark by rakesh-sankar

This repository is used to show-case benchmark of various PHP frameworks like YII, Zend, CodeIgnitor, CakePHP, DooPHP, etc.

This repository is used to show-case benchmark of various PHP frameworks like YII, Zend, CodeIgnitor, CakePHP, DooPHP, etc.

This test has been taken under the following configuration: (please let me know if you get real FUNNY results in other machines)

  • Ubuntu (marverik)
  • Intel(R) Core(TM)2 Duo CPU P8700 @ 2.53GHz - 32 bit processor
  • 4GB Memory
  • 30GB HDD

Dependencies

Make sure the following is installed: Apache Apache Bench CURL & You know the root-password (used to restart the APACHE).

Install

Extract or check-out the project and run the script as follows: 

sh phpf-benchmark.sh http://localhost/PHP-Framework-Benchmark
